# [FR-003] Skills & Experience Visualization

## Labels
`MUS`, `enhancement`, `skills`

## User Story
As a visitor, I want to see technical skills presented in a modern, engaging layout (not just a basic list) to grasp the developer's expertise.

## Proposed Solution

### Overview
Instead of a bulleted list, use a bento-box layout or an interactive node graph to display skills, languages, tools, and past experiences. 

### Implementation Flow
1. Create a `BentoGrid` or custom layout component.
2. Map skills into categories (Frontend, Backend, Infrastructure).
3. Add staggered reveal animations as the user scrolls into view.

### Technical Approach

```tsx
<div className="grid grid-cols-1 md:grid-cols-3 gap-4">
  <div className="col-span-2 bg-neutral-900 rounded-2xl p-6">
    {/* Fullstack expertise */}
  </div>
  <div className="bg-neutral-900 rounded-2xl p-6">
    {/* Tooling */}
  </div>
</div>
```

### Key Considerations
- Avoid standard skill bars (e.g., 80% HTML). Focus on actual capability and ecosystem mastery.
- Whitespace and padding must be consistent and generous.

## Acceptance Criteria
- [ ] Skills are displayed in a visually interesting layout (e.g., bento grid).
- [ ] Component is fully responsive.
- [ ] Scroll animations are subtle and performant.
